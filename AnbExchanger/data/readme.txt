The RAND Database of Worldwide Terrorism Incidents (RDWTI) Online Search Form provides tools to filter the collected list of terrorism incidents and graph the results. If, however, you want to use the RDWTI in your own statistical software, you can do so by downloading the database in its entirety using the link below.

The RDWTI is publicly available. Everyone is welcome to use our database, however all public use of these data must be attributed to the "RAND Database of Worldwide Terrorism Incidents."
